# Wind Turbine Power Calculator

A Python-based engineering tool that calculates and visualizes wind turbine power output using aerodynamic equations.

## 📌 Project Overview
This project demonstrates how Python can be used to automate engineering calculations related to wind energy systems.  
The program calculates the theoretical power output of a wind turbine based on wind speed and blade radius and visualizes the results using graphs.

## ⚙️ Features
- Calculates wind turbine power output
- Generates wind speed vs power output graph
- Exports results into CSV format for analysis

## 🧮 Formula Used
Power generated by a wind turbine:

P = 0.5 × ρ × A × v³

Where:
- **P** = Power (Watts)
- **ρ** = Air density (1.225 kg/m³)
- **A** = Swept area of the turbine blades
- **v** = Wind speed (m/s)

Swept area:

A = πr²

Where **r** is the blade radius.
